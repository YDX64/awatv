Gizlilik Politikası

Son güncelleme: 3 Mayıs 2026

AWAtv ("biz", "uygulama") senin gizliliğini önemser. Bu belge, AWAtv masaüstü ve mobil uygulamasını kullandığında hangi verilerin ne için toplandığını, nasıl saklandığını ve hangi haklara sahip olduğunu açıklar. AB Genel Veri Koruma Yönetmeliği (GDPR), Türkiye Kişisel Verilerin Korunması Kanunu (KVKK) ve eşdeğer mevzuat ile uyumludur.

1. Veri sorumlusu

AWA Digital Interactive
İletişim: support@awatv.com

2. Topladığımız veriler

2.1 Yerel cihaz verileri (her zaman)

  • IPTV listelerin (M3U / Xtream / Stalker URL'leri, kullanıcı adı, şifre) — sadece senin cihazında, şifrelenmiş Hive deposunda saklanır.
  • İzleme geçmişi & favoriler — tamamen yerel.
  • Uygulama tercihleri (dil, tema, oynatıcı ayarları) — yerel.

2.2 Hesap oluşturursan (opsiyonel)

  • E-posta adresin — Supabase üzerinde hesap oluşturmak için.
  • Şifre — bcrypt ile hash'lenir, asla düz metin olarak saklanmaz.
  • Cihaz parmak izi (anonim UUID) — çoklu cihaz senkronizasyonu için.
  • Yedeklenen veriler: IPTV listelerinin başlıkları, favoriler, izleme geçmişi. Liste içerikleri (ör. M3U URL'leri) şifrelenmiş şekilde aktarılır.

2.3 Telemetri (yalnızca onay verirsen)

  • Çökme raporları (Crashlytics) — uygulama beklenmedik şekilde kapanırsa: stack trace, OS sürümü, cihaz modeli, AWAtv sürümü. IP, e-posta veya kullanım davranışı paylaşılmaz.
  • Anonim kullanım istatistikleri (Analytics) — hangi ekranların ziyaret edildiği, hangi özelliklerin kullanıldığı. Cihaz başına anonim Firebase Installation ID ile gönderilir; kişisel veri içermez.

Bu iki kanal varsayılan olarak kapalıdır. Onboarding sırasında ayrı ayrı açabilir veya hepsini reddedebilirsin. İstediğin zaman Ayarlar → Gizlilik üzerinden değiştirebilirsin.

3. Veri işleyicilerimiz

  • Supabase (Frankfurt, EU) — kimlik doğrulama + bulut yedeği. Politikası
  • Firebase / Google LLC — Crashlytics + Analytics (yalnızca onay verirsen). Politikası
  • Cloudflare — web build + CDN. Politikası
  • GitHub — masaüstü güncellemeleri (binary indirmeleri). Politikası
  • TMDb — film/dizi metadata aramaları (sadece sen TMDb anahtarı eklediğinde). Politikası

4. Saklama süreleri

  • Yerel veriler: uygulamayı kaldırınca silinir.
  • Supabase yedeklerin: hesap kapatma talebine kadar.
  • Crashlytics: 90 gün.
  • Analytics: 14 ay (Google'ın varsayılan ayarı).

5. Haklarınız (GDPR Art. 15–22 / KVKK Md. 11)

  • Erişim: bizden hangi verilerinin tutulduğunu sorma hakkı.
  • Düzeltme: yanlış verileri güncelleme hakkı.
  • Silme: hesabını ve tüm yedeklerini kalıcı olarak silme hakkı ("unutulma hakkı").
  • Taşınabilirlik: tüm verilerini JSON formatında dışa aktarma hakkı.
  • İtiraz / kısıtlama: telemetri toplamayı reddetme hakkı.

Bu haklarını kullanmak için support@awatv.com adresine e-posta gönder — 30 gün içinde yanıt veririz.

6. Değişiklikler

Bu politikayı güncellersek, uygulama içinde yeni sürümü onaylamadan önce sana bildiririz. Tarihsel sürümlere git logundan ulaşabilirsin.

AWAtv v0.5.6 ile yayınlandı. Açık kaynak değildir; ürün dağıtımı GitHub Releases üzerinden yapılır.

Privacy Policy

Last updated: 3 May 2026

AWAtv ("we", "the app") respects your privacy. This document explains what data is collected when you use the AWAtv desktop and mobile app, why, how it is stored, and what rights you have. It complies with the EU General Data Protection Regulation (GDPR), Türkiye's KVKK, and equivalent legislation.

1. Data controller

AWA Digital Interactive
Contact: support@awatv.com

2. What we collect

2.1 Local device data (always)

  • Your IPTV playlists (M3U / Xtream / Stalker URLs, username, password) — stored locally only, in encrypted Hive boxes.
  • Watch history & favourites — fully local.
  • App preferences (locale, theme, player settings) — local.

2.2 If you create an account (optional)

  • Your email — to create the Supabase account.
  • Your password — bcrypt-hashed, never stored as plain text.
  • Device fingerprint (anonymous UUID) — for multi-device sync.
  • Backed-up data: titles of your IPTV playlists, favourites, watch history. List contents (e.g. M3U URLs) are transferred encrypted.

2.3 Telemetry (only with explicit consent)

  • Crash reports (Crashlytics) — when the app crashes: stack trace, OS version, device model, AWAtv version. IP / email / behaviour are not forwarded.
  • Anonymous usage statistics (Analytics) — which screens are visited, which features are used. Sent under an anonymous Firebase Installation ID, no PII.

Both channels default to OFF. You enable them granularly during onboarding or reject all. You can change either at any time under Settings → Privacy.

3. Our data processors

  • Supabase (Frankfurt, EU) — auth + cloud backup. Policy
  • Firebase / Google LLC — Crashlytics + Analytics (only if you opt in). Policy
  • Cloudflare — web build + CDN. Policy
  • GitHub — desktop updates (binary downloads). Policy
  • TMDb — film/series metadata lookups (only when you set a TMDb key). Policy

4. Retention

  • Local data: deleted when you uninstall.
  • Supabase backups: until you request account deletion.
  • Crashlytics: 90 days.
  • Analytics: 14 months (Google default).

5. Your rights (GDPR Art. 15–22)

  • Access: ask us what data we hold on you.
  • Rectification: correct inaccurate data.
  • Erasure: permanently delete your account and all backups ("right to be forgotten").
  • Portability: export all your data in JSON format.
  • Object / restrict: refuse telemetry collection.

To exercise these rights, email support@awatv.com — we respond within 30 days.

6. Changes

If we update this policy, the app prompts you to acknowledge the new version before continuing. Historical versions are visible in the git log.

Published with AWAtv v0.5.6. The product is closed-source; binaries are distributed via GitHub Releases.
